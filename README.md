# Manual Técnico: Diccionario de Datos
Bienvenido al anexo digital de la tesis. En este espacio interactivo se despliega el diccionario completo de campos y tablas que componen la base de datos del sistema web.

Utilice la **barra de búsqueda** de la izquierda para localizar rápidamente cualquier entidad, atributo o tipo de dato.
